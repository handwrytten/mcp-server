Handwrytten MCP Server

Send real, robot-penwritten greeting cards and notes directly from your AI assistant.

https://mcp.handwrytten.com/mcp

What is this?

The Model Context Protocol (MCP) is an open standard that lets AI assistants connect to external tools and services. This server implements MCP for Handwrytten, the leading platform for sending real handwritten notes at scale using robots with real pens.

Once connected, your AI assistant (Claude, ChatGPT, or any MCP-compatible client) can browse card designs, compose personalized messages, manage your address book, and send physical handwritten cards — all through natural conversation.

Getting Started

Option 1: Claude.ai / Remote (OAuth)

The easiest way to get started. Add Handwrytten from the MCP Directory or connect directly using the endpoint URL: 

https://mcp.handwrytten.com/mcp

You will be prompted to sign in with your Handwrytten account via OAuth. No API key needed.

Option 2: Claude Desktop / CLI (stdio)

For local use with Claude Desktop or Claude Code, install the server via npm and configure it with your Handwrytten API key.

  1. Get your API key at handwrytten.com/api
  2. Add to your Claude Desktop config (claude_desktop_config.json):
{
  "mcpServers": {
    "handwrytten": {
      "command": "npx",
      "args": ["-y", "handwrytten-mcp-server"],
      "env": {
        "HANDWRYTTEN_API_KEY": "your-api-key-here"
      }
    }
  }
}

Available Tools

The server exposes 30+ tools organized by category. Your AI assistant discovers these automatically once connected.

Sending Notes

send_order Send a handwritten card to a recipient
get_order Look up an order by ID
list_orders List recent orders with status
list_past_baskets View previously sent basket batches

Browsing Cards & Fonts

list_cards Browse available card designs
list_card_categories List card categories (Birthday, Thank You, etc.)
get_card_details Get full details for a specific card
list_fonts List available handwriting fonts
list_customizer_fonts List fonts for card cover customization

Address Book

list_recipients List saved recipient addresses
add_recipient Add a new recipient
update_recipient Update an existing recipient
delete_recipient Remove a recipient
list_senders List return addresses
add_sender Add a new return address
delete_sender Remove a return address
list_countries List supported countries
list_states List states/provinces for a country

Extras

list_gift_cards Browse gift card inserts
list_inserts List available card inserts
list_signatures List saved signature images
calculate_targets Calculate targeting info for campaigns
get_user Get current user account info

Custom Cards

list_custom_card_dimensions Available custom card sizes
upload_custom_image Upload an image for a custom card
create_custom_card Create a card from uploaded images

QR Codes

create_qr_code Generate a QR code for cards
list_qr_codes List your saved QR codes
delete_qr_code Delete a QR code
list_qr_code_frames List available QR code frame styles

Basket (Multi-Step Orders)

basket_add_order Add an order to the basket
basket_list View current basket contents
basket_count Get number of items in basket
basket_remove Remove an item from the basket
basket_clear Clear all basket items
basket_send Send all items in the basket

Interactive Apps

Typical Workflow

1

Browse Cards & Fonts

Ask your AI to show you available card designs and handwriting styles. Use the interactive card browser for a visual experience.

2

Compose Your Message

Tell your AI what you want to say. Preview how it will look in your chosen handwriting font before sending.

3

Send Your Card

Provide the recipient address and confirm. Your handwritten card will be mailed via USPS within 1-2 business days.

Important Notes

  • Always browse before sending. Use list_cards and list_fonts to discover available options before placing an order.
  • Confirm before ordering. Sending a card costs money and triggers physical mail. Your AI assistant will confirm details before placing any order.
  • Address requirements. Recipient addresses must include a valid street address, city, state, and ZIP code. PO Boxes are supported.
  • Basket for bulk orders. Use the basket tools to queue up multiple cards and send them all at once.
  • International shipping. Cards can be sent to many countries. Use list_countries to see supported destinations.
